## Setup

It is as simple as adding the following code to your Hono app.

```ts
import { Hono } from "hono";
import { HonoComment } from "@fuma-comment/server/hono";

const app = new Hono();

HonoComment({
    app,
    storage: // storage adapter
    auth: // auth adapter
});
```
